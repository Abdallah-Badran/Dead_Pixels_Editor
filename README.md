# Dead_Pixels_Editor
An image processing application that performs some operations on images and videos

## Tools used:
* Python
* tkinter
* OpenCV 

## Python Packeges
* Python
* tkinter
* OpenCV
* PIL
* moviepy
* numpy
* datetime

## Run The project
1- install the packages<br/>
2- Run the Editor.py  file<br/>
3- Watch the demo vedio to apply the CBIR and CBVR algorithms<br/>
